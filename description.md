# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
**Canva** (https://www.canva.com)

## About the Design
This infographic was created using Canva's free plan. It is a single-page visual resource designed to help first-year B.Tech students understand what Digital Literacy means.

### Topics Covered
1. **What is Digital Literacy?** — Definition and why it matters for college students
2. **Safe Internet Practices** — Strong passwords, 2FA, recognising phishing links, HTTPS
3. **Professional Online Presence** — GitHub, LinkedIn, and why to set them up early

## Design Choices
- Colour scheme: Blue and white for a clean, academic feel
- Layout: Three-column structure with icons and short bullet points
- Font: A bold sans-serif header with a clean body font for readability

## File
The exported infographic PNG is also included in this folder as `infographic.png`

## Reflection
The most challenging part was deciding what to leave out — digital literacy covers a vast range of topics. I learned that good infographic design is as much about editing as it is about adding content. Visual hierarchy (which element the eye reads first) was something I had to consciously think about for the first time.
