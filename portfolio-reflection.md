# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
GitHub is a platform for hosting and sharing code using version control (Git). It allows developers to collaborate on projects, track changes, and showcase their work publicly. I created my profile README repository (a special repo named after my username) with my name, branch, year, and a short note about what I hope to learn. GitHub will be essential for storing my projects, contributing to open-source work, and building a public coding portfolio throughout my degree.

### 2. LinkedIn (linkedin.com)
LinkedIn is the world's largest professional networking platform, used by students, job seekers, recruiters, and companies. I filled in the Education section with my current degree (B.Tech in Computer Science Engineering – AI), institution (VIT Bhopal University), and expected graduation year (2029). LinkedIn will help me connect with industry professionals, discover internship and placement opportunities, and build a professional brand online.

### 3. Kaggle (kaggle.com)
Kaggle is a platform for data science and machine learning, hosting datasets, competitions, and online courses. Since my branch is CSE with AI, Kaggle is directly relevant to my academic and career interests. I created my account and explored the beginner learning paths available. Over the next four years, I plan to participate in competitions, complete micro-courses, and use my Kaggle profile to demonstrate hands-on data skills to potential employers.

## Reflection

Setting up these three platforms in one sitting gave me a clear picture of what my digital professional identity currently looks like — and how much room there is to grow. I found LinkedIn slightly intimidating at first because it felt very "adult" and professional, but once I started filling in my Education section, I realised how easy it is to start small and build over time. The most useful insight from this task is that the earlier you start building these profiles, the more complete and impressive they look by the time you actually need them for internships or placements.
